# Spotify Listening Trends

A data storytelling project analyzing my personal Spotify listening habits using Python, Pandas, and the Spotify API.

## Project Goals
- Understand my listening behavior over time  
- Explore mood, energy, and genre patterns  
- Visualize trends using clean, UX‑friendly charts  
- Apply data cleaning, analysis, and clustering techniques  

## Project Structure
- `data/` — raw + cleaned data  
- `notebooks/` — analysis notebooks  
- `src/` — helper scripts  
- `charts/` — exported visualizations  

## Key Insights (examples)
- My listening peaks around 10 PM  
- I listen to higher‑energy music during workdays  
- My top genres shift seasonally  

## Tools Used
Python, Pandas, Seaborn, Matplotlib, Spotify API
